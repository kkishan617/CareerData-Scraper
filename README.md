# CareerData-Scraper

## Project Overview

CareerData-Scraper is a Python-based web scraping project that extracts job listing information from a public job portal and stores it in a structured CSV dataset.

## Features

* Extracts Job Titles
* Extracts Company Names
* Extracts Job Locations
* Creates a structured dataset
* Exports data to CSV format

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Pandas

## Project Structure

CareerData-Scraper/

├── job_scraper.py

├── jobs.csv

└── README.md

## Output

The project generates a CSV file containing:

* Job Title
* Company Name
* Location
